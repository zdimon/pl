# Начало проекта.

Инициализация гит репы.

    git init

    echo "# pl" >> README.md
    git init
    git add README.md
    git commit -m "first commit"
    git remote add origin git@github.com:zdimon/pl.git
    git push -u origin master

Создание ветки
    
    git branch name

Переключение на ветку 

    git checkout name

Генерация ключей

    ssh-keygen

    cat ~/.ssh/id_rsa.pub

Клонирование непозитория

    git clone 

Обновить репу.

    git pull

Активирование ВО

    . ./venv/bin/activate

Установка

    pip install name


Старт проекта 

    dgango-admin startproject name

Создание БД

    ./manage.py migrate

Запуск сервера разработки.

    ./manage.py runserver
