Frontend.

tutorial

Angular and multiproject workspase.

Creating an empty workspace.

ng new sdat --createApplication=false --directory=frontend

Change the folder and generate a new web app.

cd frontend
ng generate application web

Run the server.

ng serve web

Create a new mobile app.

ng generate application mobi

Initiate a config file for the Ionic

ionic init --multi-app

Add the Ionic lib to the mobi app.

ng add @ionic/angular --project=mobi

Add others dependencies.

  "dependencies": {
    "@ionic/angular": "^5.1.1",
    "@ionic-native/core": "^5.0.7",
    "@ionic-native/splash-screen": "^5.0.0",
    "@ionic-native/status-bar": "^5.0.0",

Create somewere a new Ionic app with the start template.

ionic start myApp sidemenu

Replace the folder app in the mobi/src with the same folder in the myApp.

Create a new library to share functionality between web and mobi apps.

ng generate library shareLib --prefix=sl

Run the builder with watcher

ng build shareLib --watch
